# Design Guidelines

Purpose: define the visual system and rules for `{{PRODUCT_NAME}}` so layouts and components stay consistent.

## Template
### Brand tone
- `{{BRAND_TONE}}`
- Design references: `{{DESIGN_REFERENCES}}`

### Tokens
- Color palette: primary, secondary, neutral, semantic
- Typography scale: font families, sizes, weights
- Spacing scale: 4/8/12/16/24/32/48 (or custom)
- Radii, shadows, borders

### Typography
- Headings: H1/H2/H3 usage rules
- Body: default size/line-height
- UI text: labels, buttons, captions

### Components
- Buttons: variants and usage rules
- Forms: inputs, validation, helper text
- Cards: spacing, density, headers/footers
- Tables or lists: layout and empty states

### Layout rules
- Grid system and breakpoints
- Max content width
- Section spacing

### Accessibility
- Contrast requirements
- Focus states
- Keyboard navigation

### Default direction
- Style: Linear-ish minimal
- Emphasis: clean spacing, sharp hierarchy, restrained color

## Example
Brand tone
- Modern, calm, trustworthy
- References: Linear, Vercel, Raycast

Tokens
- Primary: #1E1E1E, Accent: #3B82F6, Neutral: #F5F5F5
- Font: `{{PRODUCT_NAME}} Sans` (placeholder)

Components
- Buttons: primary, secondary, ghost
- Forms: inline validation with helper text
